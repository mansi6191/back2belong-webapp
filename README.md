# 🏠 Back2Belong — Campus Lost & Found Platform

A real-time web app helping campus students 
report lost items and reconnect with belongings.

## 🌐 Live Website
https://back2belong.web.app

- Mansi Vaishya(mansi6191)

## ✨ Features
- 🔐 Login with Email or Google
- 📝 Post lost & found items with photos
- 🔍 Search & filter by category/location/date
- 💬 Real-time in-app chat with item owners
- 📧 Email notifications via EmailJS
- 🔔 In-app notification system
- 🔖 Bookmark posts to track later
- 🌙 Dark mode
- 📱 Fully responsive (mobile friendly)
- 🤝 Smart matching between lost & found posts

## 💻 Tech Stack
- HTML, CSS, JavaScript
- Firebase (Auth + Firestore + Hosting)
- Cloudinary (image storage)
- EmailJS (email notifications)
